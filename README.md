# Poke Arena / FAUNWILD (Telegram Games)

Пиксельная RPG «как на Game Boy Color»: ходьба (стрелки / WASD / D-pad на телефоне), трава, ловля Kin, бои, залы, день/ночь.

## Источник

Основа — открытый проект **[FAUNWILD](https://github.com/Meliwat/faunwild)** (MIT License).  
Это **оригинальные** существа и мир (не Nintendo Pokémon). Полный клон Red/Blue с их спрайтами мы делать не можем — это копирайт.

Live demo автора: https://meliwat.github.io/faunwild/

## Управление

| Действие | Клавиши | Тач |
|----------|---------|-----|
| Ходьба | стрелки / WASD | D-pad |
| A (ок / говорить) | Z / Space | A |
| B (отмена / бег) | X | B |
| Меню | Enter | START |

Кнопка **«Счёт → TG»** — отправить прогресс в Telegram Games.
